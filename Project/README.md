##文件夹的作用
1.Project 所有的文件都保存在这个文件夹中
2.images 项目中所有应用的图片的文件夹
3.font 项目中应用到的字体
4.css 项目中的样式文件
5.js 项目中的行为文件

##功能实现
1.轮播图 通过原生js实现的轮播图，点击左右按钮实现banner图片的左右翻转，不能实现自动跳转
2.选项卡 使用原生js实现的选项卡 点击相应的主页面实现对应的副页面的跳转
3.登录注册页面 使用本地localstorage实现保存记录
